# Backend - SaaS Admin Dashboard

> **Status:** To be built next week

## Planned Tech Stack

- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication (jsonwebtoken + bcryptjs)
- CORS + dotenv

## Planned API Routes

### Auth

- `POST /auth/login`

### Clients

- `GET /clients`
- `POST /clients`
- `PUT /clients/:id`
- `DELETE /clients/:id`

### Services

- `GET /services`
- `POST /services`
- `PUT /services/:id`
- `DELETE /services/:id`

### Tickets

- `GET /tickets`
- `POST /tickets`
- `PUT /tickets/:id`
